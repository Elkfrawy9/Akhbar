## simple app to get news from API with some features

i'm working on adding new features


## Languages, libraries and tools used
- Kotlin
- Retrofit to get news and used gson converter
- Room to save specific article
- Koltin Coroutines
- Navigation graph
- dagger- Hilt
- Glide
- MVVM
